# Persona AI

A Next.js UI for chatting with two custom AI personas. Inspired by a playful, infographic-style design with a sunny yellow background, white cards, and bold accent colors.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## What's included

- **Persona selection** — Two placeholder persona cards (Persona One & Persona Two)
- **Chat interface** — Message bubbles, typing indicator, reset chat, and send input
- **Design system** — Yellow background, step badges, sticker icons, and squiggly arrows matching the reference infographic
